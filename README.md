# Smart-Parking-System

REQUIREMENTS:
- ESP32 microcontroller, ultrasonic sensor
- platformIO vscode extension
- react-vite extenstion

1) create a platformIO project
2) put files in the project
3) change the port in platformio.ini to match the correct port on your device
4) build and upload the code into the esp32 using the arrow -> at the bottom of the vscode client
5) move into the /SmartParking/ directory
6) RUN IN THE TERMINAL:
        npm install
        npm install react-router-dom
        npm run dev
7) click the link in the terminal
8) make sure your device is connected and click "connect usb" button on the top right, click your device and click connect
9) everything is set up, move your hand close to the sensor to occupy the spot.
